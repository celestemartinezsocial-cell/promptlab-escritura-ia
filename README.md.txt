# PromptLab - Escritura con IA

Herramienta interactiva para enseñar ingeniería de prompts a profesionales.

## Características

- ✅ 4 pasos guiados de aprendizaje
- ✅ Generación de contenido con IA real
- ✅ Análisis y reflexión estructurada
- ✅ Comparación de resultados
- ✅ Exportación de proyectos

## Despliegue

Este proyecto está listo para desplegarse en:
- Vercel
- Netlify
- GitHub Pages
- Cualquier hosting estático

## Uso

1. Abre `index.html` en tu navegador
2. Completa los campos del prompt
3. Genera contenido con IA
4. Analiza y mejora iterativamente
